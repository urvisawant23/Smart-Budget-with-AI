#  Health Insight & Support System

An AI-powered medical support web application built using Machine Learning and Streamlit.

This system analyzes user-selected symptoms, predicts possible health conditions, provides supportive insights, calculates BMI, and helps users find nearby medical facilities.



##  Features

- Symptom-based disease prediction using Machine Learning
- Top 3 predicted conditions with probability scores
- Severity level indicator (Low / Moderate / High)
- Detailed disease information section
- BMI Calculator (Height & Weight based)
- Nearby hospital finder using OpenStreetMap APIs
- Downloadable PDF health report
- User history logging
- Light & Dark mode compatible professional UI



##  Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas & NumPy
- OpenStreetMap (Nominatim API)
- Overpass API
- ReportLab (PDF generation)



##  How It Works

1. User enters personal details.
2. Selects symptoms from the list.
3. The trained ML model predicts possible conditions.
4. The system displays:
   - Top 3 conditions
   - Probability percentages
   - Severity indicator
   - Basic care suggestions
5. Users can:
   - Calculate BMI
   - Find nearby hospitals
   - Download a personalized health report
